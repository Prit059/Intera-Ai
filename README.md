## 🚀 What is intera.ai?

**intera.ai** is a **dual-sided SaaS platform** that combines:

* 🎙️ Voice AI Interviews
* 👁️ Real-Time Behavioral Analysis
* 📊 Smart Candidate Evaluation

It helps both **students and companies** in one unified ecosystem.

| --------------------------- | -------------------------------- |
| Practice real AI interviews | Automate candidate screening     |
| Improve confidence          | Reduce HR workload               |
| Get behavioral feedback     | Hire better candidates faster    |
| Learn anytime, anywhere     | Filter only qualified candidates |

## ❌ Problem Statement

* Students lack real interview experience
* Language barriers reduce confidence
* HR teams manually screen thousands of resumes
* No objective way to measure interview readiness
* Time wasted on unqualified candidates

---

## 💡 Our Solution

### 🔥 Core Features

* 🎙️ **AI Voice Interviews** (real-time conversation)
* 👁️ **Face + Emotion Tracking** (confidence, eye contact, nervousness)
* 🧠 **Adaptive Questioning Engine**
* 📊 **Behavioral Scoring Dashboard**
* 📝 **AI-Based Aptitude Tests**
* 🎥 **Interview Recording & Playback**
* 🌐 **Multilingual Interviews**
* 🔒 **Privacy-First (on-device video processing)**
* 📬 **Auto Candidate Filtering for HR**

---

## 🏗️ Tech Stack

### 🎙️ Voice AI Layer

* Vapi AI (low latency < 500ms)
* Dynamic conversation engine
* Real-time interaction

### 👁️ Vision AI Layer

* MediaPipe Face Mesh (478 points)
* Emotion + behavior tracking
* Runs fully on-device

### 🌐 Backend & Platform

* MERN Stack (MongoDB, Express, React, Node)
* Scalable SaaS architecture

---

## 🔄 How It Works

### 👨‍🎓 For Students

1. Sign up & choose interview type
2. AI starts real-time interview
3. Face & behavior tracked silently
4. Questions adapt dynamically
5. Get full performance report
6. Improve & retry

---

### 🏢 For Companies

1. Create job role & requirements
2. Generate AI interview + aptitude test
3. Candidates take test remotely
4. AI evaluates + scores candidates
5. Only qualified candidates sent to HR
