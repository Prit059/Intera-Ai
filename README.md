## 🔄 How It Works

### For Students:

1️⃣ Sign up & select interview type (Technical / HR / Behavioral)
2️⃣ AI interviewer begins a real-time voice conversation
3️⃣ MediaPipe silently analyzes facial behavior throughout
4️⃣ AI adapts questions based on your responses
5️⃣ Session ends → Receive detailed performance report
6️⃣ Review behavioral metrics: confidence, eye contact, nervousness
7️⃣ Practice again → Track improvement over time

- **Node.js** v18+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/intera.ai.git
cd intera.ai

# Install backend dependencies

# Install frontend dependencies

npm install
